
To run this simulation locally:
	You must have python installed.
	Open a command prompt in the root folder (The folder that has index.html)
	
	If you have python 3:
		execute "python -m http.server" without the quotes in the command prompt
	If you have python 2:
		execute "python -m SimpleHTTPServer" without the quotes in the command prompt

	Go to a web browser and type in "localhost:8000" into the address bar without the quotes
